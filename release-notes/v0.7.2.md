# v0.7.2

## 📝 Documentation Enhancements (English)

Expanded practical guides and API references to make it easier to adopt WarpVector's auto-learning and federated learning features.

- **New Documentation**: Added a practical guide "Project Integration for Auto-Learning and Federated Learning" (`docs/cookbook/practical-auto-learning.ja.md`).
  - Technical background on Affine transformations, LoRA, and WASM.
  - Initial cold-start training process using `IntentMatrixFactory`.
  - Simplest ways to integrate a feedback API into existing projects using `FeedbackCollector` and `AdaptiveScheduler`.
  - Explanation of secure federated learning using `FederatedAggregator`.
  - Best practices for hyperparameter tuning (`batchSize`, `temperature`, etc.).
- **API Reference Update**: Added the following auto-learning related classes to `docs/api-reference.ja.md`:
  - `IntentMatrixFactory`
  - `FeedbackCollector`
  - `AdaptiveScheduler`
  - `FederatedAggregator`
  - Related type definitions (`IntentMatrixFactoryOptions`, `FeedbackCollectorConfig`, `SchedulerConfig`, etc.)

## 🔧 Internal Changes (English)
- Fixed a failing test in `AnomalyDetectionAdapter` to properly handle `NaN` inputs in safe mode. (No functional changes in core, documentation and tests only).
