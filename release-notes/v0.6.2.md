# Release Notes - v0.6.2

## 🇬🇧 English

### 🚀 Features & Refactors
- **core**: Simplified the `WarpPipeline` step registration process, eliminating the need to specify redundant name arguments.

### 🐛 Bug Fixes
- **gpu**: Fixed an issue where `bun:test` types were not loaded due to `tsconfig.json` settings, normalizing type resolution in the testing environment.

### 📖 Documentation
- Implemented a major update to the README.
  - Added architectural diagrams.
  - Expanded feature highlights and primary use cases.
  - Organized installation instructions for all packages.
- Updated code examples: Reflected the rename of the `tune` method to `encode` throughout the documentation.
- Updated explanations regarding advanced features such as Task Arithmetic.

### 📦 Maintenance
- Synced and updated the versions of all workspace packages to `0.6.2`.
