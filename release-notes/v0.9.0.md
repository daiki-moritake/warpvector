# v0.9.0 — Package Reorganization & Stable Promotions

## 🎯 Highlights

This release focuses on **package hygiene and developer experience improvements**. All features previously available only through `@warpvector/experimental` are now accessible directly from the root `warpvector` package, and the experimental package has been formally deprecated.

## ✨ New Features

### Rerank Exports Available from Root Package
`ColbertAdapter`, `TimeReversalReranker`, `MultipathScatteringReranker`, and `BaseGraphReranker` are now re-exported from the root `warpvector` package:

```typescript
// Before (v0.8.x): Required separate import
import { ColbertAdapter } from "warpvector/rerank";

// After (v0.9.0): Direct import from root
import { ColbertAdapter, TimeReversalReranker } from "warpvector";
```

## ⚠️ Deprecations

### `@warpvector/experimental` Package Deprecated
All features have been stable in `@warpvector/extras` and `@warpvector/rerank` for multiple releases. The experimental package now emits a runtime deprecation warning and will be removed in v1.0.0.

```diff
- import { ColbertAdapter } from "@warpvector/experimental";
+ import { ColbertAdapter } from "warpvector";

- import { VsaAdapter, AnomalyDetectionAdapter } from "@warpvector/experimental";
+ import { VsaAdapter, AnomalyDetectionAdapter } from "warpvector";
```

## 🔧 Internal Changes
- Updated `@warpvector/extras` description and keywords to accurately reflect its contents (removed ColBERT reference, added Anomaly Detection)
- Updated test badge in README from 297 to 330 (reflecting actual test count)

## ✨ 新機能

### Rerank のエクスポートがルートパッケージから利用可能に
`ColbertAdapter`、`TimeReversalReranker`、`MultipathScatteringReranker`、`BaseGraphReranker` がルートの `warpvector` パッケージから re-export されるようになりました：

```typescript
// 以前 (v0.8.x): 別途インポートが必要
import { ColbertAdapter } from "warpvector/rerank";

// 今後 (v0.9.0): ルートから直接インポート
import { ColbertAdapter, TimeReversalReranker } from "warpvector";
```

## ⚠️ 非推奨化

### `@warpvector/experimental` パッケージの非推奨化
すべての機能は `@warpvector/extras` と `@warpvector/rerank` で複数のリリースにわたり安定的に提供されています。experimental パッケージはランタイムで非推奨警告を出力するようになり、v1.0.0 で完全に削除される予定です。

```diff
- import { ColbertAdapter } from "@warpvector/experimental";
+ import { ColbertAdapter } from "warpvector";

- import { VsaAdapter, AnomalyDetectionAdapter } from "@warpvector/experimental";
+ import { VsaAdapter, AnomalyDetectionAdapter } from "warpvector";
```
