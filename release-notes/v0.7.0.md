# Release Notes - v0.7.0

## 🇬🇧 English

### 🚀 Features & Refactors
- **core**: Introduced `AlignmentAdapter`, a specialized adapter for zero-downtime vector database migration. It allows querying an old vector space (e.g., `ada-002`) using a new embedding model (e.g., `text-embedding-3-small`) without re-indexing the database.
- **train**: Introduced `MigrationTrainer`, a dedicated trainer that uses an Adam optimizer to learn the optimal affine transformation (alignment matrix) between two different embedding models using a small set of parallel data.
- **playground**: Added a new interactive usecase demo ("Zero-Downtime Model Migration") to visually demonstrate vector alignment.

### 📖 Documentation
- Published a new Zenn article detailing the Zero-Downtime Migration architecture.
- Re-focused the existing Zenn article strictly on Privacy and Federated Learning for Edge AI.
- Updated the primary READMEs to highlight the "Zero-Downtime Migration" as a core capability.
- Renamed and overhauled the `10-projection-migration.md` documentation to `10-alignment-and-migration.md` to reflect the new `AlignmentAdapter`.

### 📦 Maintenance
- Synced and updated the versions of all workspace packages to `0.7.0`.

## 🇯🇵 日本語

### 🚀 Features & Refactors
- **core**: ゼロダウンタイムでのベクトルDB移行（モデル切り替え）に特化した新しいアダプター `AlignmentAdapter` を導入しました。これにより、DBの再インデックスを行うことなく、新しい埋め込みモデル（例：`text-embedding-3-small`）を使って古いベクトル空間（例：`ada-002`）を検索できるようになります。
- **train**: 新旧モデルのペアデータ（100件程度）から、2つのベクトル空間を翻訳する最適な変換行列を学習する専用の `MigrationTrainer` を導入しました。
- **playground**: ブラウザ上でベクトルのアラインメント（空間翻訳）を視覚的に体験できる新しいインタラクティブ・デモ（"ゼロダウンタイム・モデル移行"）を追加しました。

### 📖 Documentation
- ゼロダウンタイム・モデル移行のアーキテクチャに特化した新しいZenn記事を公開しました。
- 既存のZenn記事を「エッジAIにおけるプライバシー保護と連合学習」に完全にフォーカスするよう構成を見直しました。
- プロジェクトの README において「ゼロダウンタイム移行」を主要機能としてハイライトしました。
- 旧 `10-projection-migration.md` ドキュメントを `10-alignment-and-migration.md` にリネームし、`AlignmentAdapter` を用いたワークフローに全面改訂しました。

### 📦 Maintenance
- ワークスペース内すべてのパッケージのバージョンを `0.7.0` に同期・アップデートしました。
